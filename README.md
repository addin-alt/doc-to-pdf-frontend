# DOC to PDF Frontend

This is the **frontend** of the Doc-to-PDF application, which allows users to upload DOCX files and convert them into PDF format. The frontend is designed to be simple, responsive, and user-friendly.

## Demo

You can try the live frontend here: [Frontend Demo](https://addin-alt.github.io/doc-to-pdf-frontend/)  
**Backend API**: [Doc-to-PDF Backend](https://doc-to-pdf-backend.onrender.com)

## Features

- Upload DOCX files and convert them to PDF
- Simple, clean user interface
- Responsive design for mobile and desktop
- Preview of uploaded files before conversion
- Integration with backend API for PDF generation

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Axios / Fetch API for HTTP requests
